# 🚚 Amazon Delivery Time Prediction App

A machine learning web application built with Streamlit that predicts delivery times based on various factors including agent details, environmental conditions, and order characteristics.

## 🎯 Features

- **Real-time Predictions**: Get instant delivery time estimates
- **Interactive Interface**: User-friendly web interface built with Streamlit
- **Advanced ML Model**: Uses Gradient Boosting Regressor (RMSE: 25.80 minutes)
- **Multiple Factors**: Considers weather, traffic, distance, agent rating, and more

## 🚀 Live Demo

**App URL:** https://your-app-name.streamlit.app

## 📊 Model Performance

- **RMSE:** 25.80 minutes
- **MAE:** 20.12 minutes  
- **R² Score:** 0.750
- **Training Data:** 43,000+ delivery records

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **ML Model:** Scikit-learn (Gradient Boosting Regressor)
- **Data Processing:** Pandas, NumPy
- **Deployment:** Streamlit Community Cloud

## 📋 Input Features

- Agent Age & Rating
- Distance (km)
- Weather conditions
- Traffic conditions
- Vehicle type
- Delivery area
- Product category
- Time factors (hour, day, month)
